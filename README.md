## How to Make Changes

1. Edit the files in this folder (`C:\Users\Ansh\Documents\Website`)
2. Commit and push to GitHub:
   ```
   git add .
   git commit -m "Your message"
   git push
   ```
3. Vercel auto-deploys within seconds
